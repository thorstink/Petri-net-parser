◆ registerTransitionCallback()

std::function< void()> PetriNet::registerTransitionCallback ( const std::string &  transition) const
noexcept

register transition gives a handle to manually force a transition to fire. This is usefull if you want to trigger a transition that has no input places. It is not recommended to use this for transitions with input places! This violates the mathematics of petri nets.

Parameters
transitionthe name of transition. This transition has to be available in the net.
