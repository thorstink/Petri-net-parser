Detailed Description

PolyTransition is a wrapper around any type that you want to tie to a transition. Typically this is an invokable object, such as a function, that executes some side-effects. The output of the invokable object can be used to communicate success or failure to the petri-net executor. You can create custom behavior by defining a tailored "fire(const A&)" for your transition payload.
