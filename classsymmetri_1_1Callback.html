Detailed Description

Callback is a wrapper around any type that you to tie to a transition. Typically this is an invokable object, such as a function, that executes some side-effects, but it can by anything if you implement a fire-function for it. The output can be used to communicate success or failure to the petri-net executor. You can create custom behavior by defining a tailored "Result fire(const A&)" for your class A.

Random note, this class is inspired/taken by Sean Parents' talk "inheritance is the base class of evil".
